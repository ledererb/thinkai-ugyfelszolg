Projekt neve	ThinkAI Hangalapú Showcase Asszisztens
Célja	Weboldalon elérhető AI hangasszisztens prototípus, amely bemutatja a ThinkAI képességeit potenciális ügyfeleknek
Technológiai stack	Pipecat + Deepgram + Claude + Cartesia
Infrastruktúra	Vercel (frontend) + Railway (backend)
Határidő (prototípus)	~3–4 munkanap
Becsült havi üzemeltetési költség	~$50–120/hó (forgalomtól függően)

1. Projekt célja
A ThinkAI weboldalára (thinkai.hu) egy hangalapú AI asszisztenst készítünk, amely két célt szolgál egyszerre:
•	Bemutató eszköz: a látogató azonnal megtapasztalja, milyen egy éles AI hangmegoldás — maga a widget a legjobb értékesítési érv.
•	Információs asszisztens: válaszol a ThinkAI szolgáltatásaival, projektjeivel és módszertanával kapcsolatos kérdésekre, és érdeklődő esetén a kapcsolatfelvételi űrlaphoz irányítja a látogatót.

Az asszisztens magyar nyelven kommunikál alapértelmezetten, de automatikusan átvált angolra, ha a látogató angolul szólal meg.

2. Funkcionális követelmények
2.1 Hanginterakció
•	Valós idejű kétirányú hangkommunikáció a böngészőből (nem szükséges telefonszám vagy külső alkalmazás)
•	Folyamatos audio streaming: mikrofon → szerver → AI → visszajátszás
•	Megszakításkezelés: ha a felhasználó közbeszól, az asszisztens megáll
•	Hangszemélyiség: természetes, magabiztos, üzleti hangvétel
2.2 Nyelvi viselkedés
•	Alapértelmezett nyelv: magyar
•	Automatikus nyelvfelismerés: ha a látogató angolul szólal meg, az asszisztens angolra vált
•	Rövid, lényegre törő válaszok — hangformátumhoz optimalizálva (1–3 mondat)
2.3 Tartalom & tudásbázis
•	ThinkAI szolgáltatások: egyedi AI fejlesztés, pályázati tanácsadás, EAISY termékcsalád
•	Munkamódszer: Audit → Prezentáció → Kiválasztás → Megvalósítás + pénzvisszafizetési garancia
•	Kiemelt projektek: Smart Számla Értesítő, Egészségügyi Asszisztens, Marketing Disztribútor
•	Célszektorok: gyártás, logisztika, pénzügy & jog
•	Partnerek: BDPST Koncept, Develor, WSZL, ClearService, Duna Autó
•	CTA: hello@thinkai.hu és az ajánlatkérő űrlap
2.4 Frontend widget
•	Egyszerű "Beszélj velünk" gomb a weboldalon (jobb alsó sarok)
•	Állapotjelzés: inaktív / csatlakozás / aktív / befejezve
•	Leállítás gomb
•	Hibaüzenet, ha a kapcsolat nem jön létre
•	Mikrofon-engedélykérés kezelése

3. Technikai architektúra
3.1 Rendszerábra
Böngésző (mikrofon)
    ↓  WebSocket (audio stream)
Railway szerver (Pipecat pipeline)
    ├─ Deepgram STT  (hu/en)
    ├─ Claude Sonnet (rendszerprompt)
    └─ Cartesia TTS  (magyar hang)
    ↓  WebSocket (audio stream vissza)
Böngésző (hangszóró)

3.2 Komponensek
Réteg	Eszköz / Szolgáltatás	Indoklás
Frontend	Next.js / Vercel	Meglévő infrastruktúra, GitHub push = deploy
Voice kliens	@pipecat-ai/client-web	Kész JS könyvtár, kezeli a böngésző audio-t
Backend	Pipecat + FastAPI	Teljes voice pipeline keretrendszer, Railway-en fut
STT	Deepgram Nova-2	Legjobb valós idejű latencia, magyar nyelv támogatás
LLM	Claude Sonnet 4.6	Eszközhívás, hosszú kontextus, magyar tudás
TTS	Cartesia	Alacsony latencia (~200ms), magyar hangok
Hosting	Railway	Perzisztens WebSocket, egyszerű deploy, ~$5/hó

6.2 Forgalomalapú API költségek
Modell: 20 hívás/nap × 5 perc/hívás × 20 munkanap = ~2 000 perc/hó

Szolgáltatás	Díjszabás	Havi becslés
Deepgram STT	$0.0059/perc	~$12
Claude Sonnet 4.6	$3/M input + $15/M output token	~$20–40
Cartesia TTS	$15/1M karakter	~$15–25
ÖSSZESEN (API)		~$47–77/hó
MINDÖSSZESEN		~$52–102/hó

Összehasonlítás: VAPI platform ugyanekkora forgalomnál
VAPI: $0.05/perc × 2 000 perc = $100 alap + API költségek = ~$250–350/hó
Saját megoldás: ~$52–102/hó — 3–4x olcsóbb és teljes kontroll

7. Jövőbeli fejlesztési lehetőségek
A prototípus után könnyen bővíthető a rendszer:

•	Eszközhívások: ajánlatkérési adatok mentése Supabase-be közvetlenül a hívás során
•	Calendly integráció: időpontfoglalás hangból
•	Telefonos verzió: Telnyx BYOC SIP trunk bekötésével magyar telefonszámra is kiterjeszthető
•	Többnyelvűség: automatikus nyelvfelismerés finomhangolása
•	Elemzés: hívások mentése, átirat generálás, konverziók mérése
•	Iparági verziók: ugyanez a rendszer más ügyfelek számára (fogászat, könyvelés, ügyfélszolgálat)
